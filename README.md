## 사용 플러그인

이 프로젝트는 VRM4U 플러그인 ,Kawaii Physics 플러그인 사용합니다.

## VRM4U
VRM4U는 Unreal Engine 4에서 VRM 파일을 지원하는 플러그인입니다. 자세한 내용은 [VRM4U GitHub](https://github.com/ruyo/VRM4U)에서 확인할 수 있습니다.
